how to play on mobile

1. download Pydroid 3 or any other python compiler you can find on the app store

2. open pydroid 3 and click on the file button on the top right of your phone

3. open the game and click the run button (big yellow play button)

4. enjoy (working on apk version will come soon) 

#################################################################################
how to play on desktop 

1. download python from https://www.python.org/downloads/

2. open cmd

3. navigate to the parent save location of the game script

commands for cmd to open game

cd (change directory)

python blades_of_eternity.py (to open game file)

4. enjoy (working on exe version as well)
